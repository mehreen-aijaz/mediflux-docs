The **Drug List** is your main directory of every drug in your pharmacy's inventory. It gives you a real-time snapshot of stock levels, value, and expiry status, with the flexibility to view your inventory either grouped by drug or broken down batch-by-batch.

---

## Summary cards

At the top of the page, three summary cards give you an at-a-glance overview of your entire inventory:

| Card | What it shows |
| --- | --- |
| **Total Drugs** | Total number of unique drugs in your inventory |
| **Total Value** | Total MRP value of your current stock, along with its cost value |
| **Expired / Out of Stock** | Number of drugs that have crossed their expiry date, and number of drugs with zero stock remaining |

## Searching drugs

The search bar at the top lets you search across **brands, salts, drug classes, or manufacturers**. Just start typing and the list filters instantly.

To the left of the search bar is an **All** dropdown. Use it to narrow the search scope to a specific field:

| Option | Searches by |
| --- | --- |
| **All** | Every field below, combined |
| **Batch Number** | The drug's batch number |
| **HSN** | HSN code |
| **Manufacturer** | Manufacturer name |
| **Rack Number** | Rack location |
| **Salts** | Active salts in the drug |

## Switching between Consolidated and Separated View

Below the filter chips, two tabs let you switch how drugs are displayed:

- **Consolidated View:** groups every batch of the same drug into a single row, useful for a quick, uncluttered look at your inventory by drug name
- **Separated View:** lists every batch as its own row, with rack number and full row-level actions available

## The drug table

### Consolidated View columns

| Column | Description |
| --- | --- |
| **S.No** | Row number |
| **Name** | Drug name |
| **Batch** | Batch number |
| **HSN** | HSN code. Shows `—` if not added |
| **Stock** | Units currently available |
| **Trade P** | Trade price, the wholesale cost paid to the supplier. Shows `—` if not added |
| **MRP** | Maximum retail price |
| **Expiry** | Expiry date (month/year) |
| **Supplier** | Supplier the drug was purchased from |

### Separated View columns

Separated View shows the same information as Consolidated View, with two differences: a **Rack** column instead of grouping by drug, and an **Actions** column for row-level actions.

| Column | Description |
| --- | --- |
| **Rack** | Rack location of the batch. Shows `—` if not added |
| **Price** | Same as MRP |
| **Actions** | Opens a menu with **Edit** and **Add Deal** for that drug |

## Selecting drugs and bulk actions

Selecting one or more rows using the checkboxes brings up a toolbar with bulk actions. The options available depend on the view you're in:

| Action | Available in | What it does |
| --- | --- | --- |
| **Export Activities** | Both views | Exports the activity history of the selected drug(s) |
| **Generate Debit Note** | Separated View only | Raises a debit note to the supplier for the selected drug, to request a return |
| **Reorder** | Separated View only | Sends a purchase order to the supplier for the selected drug. This only works if the supplier is also on MediFlux |
| **Update** | Separated View only | Opens a form to update the selected drug's batch number, rack number, stock, expiry, HSN code, NDC, dosage form, tax rates, salts, and drug classes |
| **Delete** | Separated View only | Removes the selected drug(s), moving them to Trash |

---

<Note>
Consolidated View only allows exporting activity for selected groups. Switch to Separated View to reorder, update, generate a debit note, or delete a specific batch.
</Note>

## Add Deal

The **Add Deal** action (from the row's **⋮ menu** in Separated View) lets you set up a purchase scheme for a drug, for example a "buy X get Y free" offer, by specifying a **Buy Quantity**, **Free Quantity**, a **Start Date** and **End Date**, and whether the deal is **Active**.

## Filter & Sort panel

Click the **Filter** button to open the **Filter & Sort** panel, with three tabs:

### Filters

| Option | What it does |
| --- | --- |
| **Quick Filters** | Low Stock, Expiring Soon, Out of Stock |
| **Sort By** | Sort by Name, Stock, Expiry, or Date Added |
| **Type** | All, NRx, Rx, H1 |
| **Stock** | In Stock, Out of Stock, or Low Stock Only |
| **Expiry** | Valid, Expired, 1 Month, 3 Months, 6 Months, 1 Year, or a Custom range |
| **Suppliers** | Filter by one or more suppliers |

## Importing your inventory

If you're setting up MediFlux for the first time or need to add drugs in bulk, the **Import** button opens a step-by-step wizard:

1. **Upload:** upload your inventory as a CSV or Excel file. You can also download a template to see the expected format
2. **Map Headers:** match the columns in your file to MediFlux's drug fields
3. **Preview:** review your data before confirming the import
4. **Complete:** finishes the import and adds the drugs to your inventory
