The **Trash** page holds all customer profiles that have been deleted from your system. Deleted customers are not permanently removed immediately. They stay in Trash so you can restore them if needed.

Navigate here from the sidebar: **Customers → Trash**.

---

## The trash table

The Trash page shows the same table structure as the Customer List in list view:

| Column | Description |
| --- | --- |
| **Name** | Customer name with type (Consumer/Business) shown below |
| **Age** | Customer's age if added. Shows `—` if not provided |
| **Gender** | Customer's gender if added |
| **Phone** | Primary contact number |
| **Email** | Customer's email address |
| **Outstanding** | Amount owed by this customer at time of deletion |
| **Last Payment** | Date of last payment. Shows `—` if none recorded |
| **Actions** | Options to manage the deleted customer |

The page shows a **Found X** count above the table.

## Row actions

Clicking the **three-dot menu (⋮)** on any row gives you two options:

| Option | What it does |
| --- | --- |
| **View Sales** | View all sales history linked to this customer even after deletion |
| **Restore** | Restores the customer profile back to your active Customer List |

<Note>
Restoring a customer brings back their full profile, sales history, and outstanding balance exactly as it was at the time of deletion.
</Note>
